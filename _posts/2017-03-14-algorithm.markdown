# 前言

**什么是算法？：**

 - 算法是把输入转换成输出的计算步骤的一个序列，是用于求解良说明的计算问题的工具；
 - 注意，若对每个输入实例，算法都以正确的输出停机，则称算法是正确的，并称算法解决的给定的计算问题。
 - 注意，不正确的算法只要其错误率可控，有时可能是有用的。
